Model-Based Testing and Internet of Things (IoT): A Literature Review

A systematic review of five recent papers on Model-Based Testing (MBT) for IoT systems.

Overview

This report reviews and synthesizes five papers covering systematic IoT testing reviews, early-safety analysis, model-based security testing, automata-learning-based protocol testing, and industry testing practices, to map the current state of MBT for IoT.

Key Highlights


Defined five research questions on MBT application, modeling formalisms/tools, limitations, and effectiveness in IoT testing.
Reviewed methodologies from a PRISMA-based review of 83 studies to an automata-learning approach that uncovered protocol faults in 4 of 5 MQTT broker implementations.
Synthesized all five papers into a comparative strengths/limitations table.
Identified the lack of standardized evaluation frameworks and reproducible testbeds as the most significant recurring gap.
Proposed future directions: unified testing frameworks, open benchmarking datasets, and stronger academia–industry integration.
